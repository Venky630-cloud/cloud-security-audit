# AuditX Cloud Security

AuditX is an AI-powered cloud security auditing platform designed to identify misconfigurations in AWS environments.

## 🚀 Features

- Detect IAM privilege escalation risks
- Identify publicly exposed Amazon S3 buckets
- Analyze insecure network configurations in Amazon VPC
- Continuous monitoring using AWS CloudTrail and AWS Config
- AI-driven remediation suggestions

## 🧱 Tech Stack

- AWS IAM
- Amazon S3
- Amazon VPC
- AWS Lambda
- AWS CloudTrail
- AWS Config
- Amazon GuardDuty
- Amazon Bedrock (for AI analysis)

## 🧪 Current Status

Currently building initial prototype focusing on:
- IAM policy analysis
- S3 exposure detection
- Basic VPC misconfiguration checks

## 🎯 Goal

To provide startups with automated, scalable, and cost-effective cloud security auditing.

## 📌 Future Plans

- Real-time alerting system
- Dashboard for security insights
- Automated remediation engine
